
# Vtuber Desktop

## 프로젝트 소개

Vtuber Desktop은 사용자의 데스크톱 환경에서 인공지능 VTuber 캐릭터를 표시하고 상호작용할 수 있도록 제작한 Unity 기반 데스크톱 애플리케이션입니다.

사용자는 데스크톱 화면 위에서 캐릭터를 확인할 수 있으며, 다양한 애니메이션과 기능을 통해 더욱 생동감 있는 환경을 경험할 수 있습니다.

## 개발 목적

최근 VTuber와 가상 캐릭터 콘텐츠의 인기가 증가하면서 사용자가 일상적으로 캐릭터와 함께할 수 있는 데스크톱 애플리케이션을 제작하고자 하였습니다.

Unity의 실시간 렌더링 기술과 캐릭터 제어 시스템을 활용하여 가상 캐릭터를 자연스럽게 표현하는 것을 목표로 개발하였습니다.

## 개발 환경

* Unity
* C#
* Python
* Ollama LLM
* Git / GitHub

## 주요 기능

* VTuber 캐릭터 표시

* 데스크톱 환경 연동
* 사용자 인터랙션 기능
* 실시간 캐릭터 상태 관리

## 개발 과정에서 해결한 문제

### 캐릭터 상태 관리

다양한 애니메이션 상태가 충돌하는 문제가 발생하였습니다.

#### 해결 방법

상태 머신 구조를 적용하여 캐릭터의 행동을 체계적으로 관리하고 애니메이션 전환을 안정적으로 구현하였습니다.

## 배운 점

* Unity 기반 애플리케이션 구조 설계
